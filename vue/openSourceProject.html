# 思维导图

# 1、Web UI 库

# 1、ElementUI Plusopen in new window

一套为开发者、设计师和产品经理准备的基于 Vue 3.0 的桌面端组件库

# 2、Ant Design of Vueopen in new window

Ant Design 的 Vue 实现，开发和服务于企业级后台产品

# 3、BalmUIopen in new window

基于谷歌的 Material Design，附带 Vue 插件和指令，以及从简单到复杂的高度可定制组件

# 4、Naive UIopen in new window

图森 Vue3 的组件库，文档完整，我项目中经常使用

# 5、arco.designopen in new window

字节跳动企业级产品设计系统，支持 React 和 Vue 双版本

# 6、Quasaropen in new window

轻松构建高性能和高质量的 Vue.js 3 用户界面，好用，但没有中文文档

# 7、iDUXopen in new window

Vue3.x 的 UI 组件库，完全使用 TypeScript 开发

# 8、TDesignopen in new window

腾讯业务团队在服务业务过程中沉淀的一套企业级设计体系 https://tdesign.tencent.com/vue-next/overview

# 9、PrimeVueopen in new window

易于使用、多功能、高性能的 Vue UI 组件库

# 10、DevUIopen in new window

华为基于 Vue3 和 DevUI 设计的 UI 组件

# 11、vuestic-uiopen in new window

Vue 3 的免费和开源 UI 库 ,UI 非常好看，并且有可用后台管理界面。

# 12、Headless UIopen in new window

完全无样式、完全可访问的 UI 组件，旨在与 Tailwind CSS 完美集成。

# 13、View UI Plusopen in new window

基于 Vue.js 3 的企业级 UI 组件库和前端解决方案

# 2、移动 UI 库

# 1、Vantopen in new window

有赞轻量、可靠的移动端组件库

# 2、NutUIopen in new window

京东风格的轻量级移动端 Vue 组件库，非常适合移动端电商使用。

# 3、Varletopen in new window

Material 风格移动端组件库 ，文档非常齐全。

# 4、nutui-bingoopen in new window

京东基于 NutUI 的抽奖组件库，助力营销活动和小游戏场景。

# 3、相关工具

# 1、unplugin-vue-componentsopen in new window

antfu 按需组件自动导入，开箱即用地支持 Vue 2 和 Vue 3，Tree-shakable，只注册你使用的组件，附有流行 UI 库的内置解析器。

# 2、vuex-persistedstateopen in new window

在页面重新加载之间保持并重载您的 Vuex 状态

# 3、vuex-persistopen in new window

支持 Typescript 的 Vuex 插件，它能够将应用程序的状态保存到持久存储中，例如 Cookies 或 localStorage

# 4、@vueuse/gestureopen in new window

手势库，使应用程序具有交互性

# 5、unplugin-auto-importopen in new window

antfu 自动导入 Vite、Webpack、Rollup 和 esbuild 的 API。支持 TypeScript。

# 6、pinia-plugin-persistedstateopen in new window

Pinia 商店的可配置持久性

# 7、TermUIopen in new window

一个基于 Vue.js 的终端 UI 框架，可让您轻松构建现代终端应用程序

# 4、可视化

# 1、Pdfvueropen in new window

DF 查看器，使用 Mozilla 的 PDF.js，支持 Vue2 和 Vue3

# 2、vue3-marqueeopen in new window

无缝滚动组件

# 3、Vue-EChartsopen in new window

百度 ECharts 的 Vue.js 组件。 配置参考 Echarts 官方 基于 ECharts v5+ 开发，适用于 Vue.js 2/3。

# 4、iconparkopen in new window

字节跳动出品，将一个 SVG 图标转化为多个主题，并生成 React 图标，Vue 图标，svg 图标

# 5、插件

# 1、vue-multiselect-nextopen in new window

Vue.js 的通用选择/多选/标记组件

# 2、vue-print-nbopen in new window

# 3、vue-i18n-nextopen in new window

Vue3 的国际化插件

# 4、vue-cropperopen in new window

简单的 vue 图片裁剪插件

# 5、Vue Grid Layoutopen in new window

Vue.js 的网格布局系统

# 6、Vue Qrcode Readeropen in new window

# 7、Makeit Captcha

# 8、vue3-clipboardopen in new window

Vue 3 的 clipboard.js

# 9、vue.draggableopen in new window

基于 Sortable.js 的 Vue 3 拖放组件

解决移动端（已支持 PC）各种滚动场景需求的插件。
